# ckhotgav 的 AI 編輯器新手全能懶人包

這個 repo 幫助完全不懂程式的新手，快速在 **Antigravity** 或 **OpenCode** 上建立一套結合「自動知識庫 + 版控 + 一鍵部署」的 AI 開發環境。

不放個人 NotebookLM 清單、研究產物、生成圖片、測試專案或任何帳號資料。

> ⚠️ **適用平台**：本懶人包以 **Windows** 為主要支援平台。macOS / Linux 用戶可參考步驟，但部分指令（如 `npm.cmd`、PowerShell）需自行轉換為對應指令。

## 目前包含的技能

| 編號 | 技能名稱 | 說明 |
|------|---------|------|
| `00` | 一鍵全裝 | 自動依序安裝所有核心技能 |
| `01` | 基礎環境 | Node.js、Python、PowerShell 權限設定 |
| `02` | NotebookLM | AI 知識庫串接與 MCP 外掛設定 |
| `03` | GitHub | 版本控制、帳號登入、避坑指南 |
| `04` | Netlify | 靜態網頁一鍵部署 |
| `05` | GAS | Google Apps Script 雲端腳本部署 |
| `06` | 專案流程自動化 | 開工/收工/里程碑 關鍵字驅動的工作流程 |
| `07` | Obsidian第二大腦 | 本地筆記庫安裝與 MCPVault 外掛設定 |
| `08` | 系統健康檢查 | 列出所有已裝工具的版本與狀態 |
| `09` | 一鍵升級 | 所有 CLI 工具一次升級到最新版 |
| `10` | 疑難排解 | 常見錯誤與修復流程 |
| `11` | 其他 AI 編輯器（選用） | Claude Code / Gemini CLI / Codex CLI 設定 |
| `12` | 其他部署平台（選用） | Vercel / Cloudflare Pages / Railway 設定 |

## 使用方式

### 方式一：直接叫 AI 幫你裝（最簡單）

把這行複製貼給你的 AI 助理：

```
這是我的 AI 編輯器懶人包 https://github.com/ckhotgav-git/ckhotgav-lazy-pack
請讀取 repo 內容，列出所有可用的技能，問我要安裝哪些，然後自動幫我安裝。
```

### 方式二：直接開啟主懶人包

1. 開啟 `AI-Editor-新手全能懶人包.md`
2. 把文件內容貼給你的 AI 助理，請它帶你一步一步設定
