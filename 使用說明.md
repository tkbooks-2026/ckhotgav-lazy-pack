# ckhotgav-lazy-pack 使用說明

## 這是什麼？
ckhotgav 整理的 **AI 編輯器新手全能懶人包**，讓完全不懂程式的新手快速在 Antigravity 或 OpenCode 上建立開發環境。

## GitHub 同步版本
- **原始來源**：https://github.com/ckhotgav-git/ckhotgav-lazy-pack
- **Fork 備份**：https://github.com/tkbooks-2026/ckhotgav-lazy-pack

## 目錄結構
```
ckhotgav-lazy-pack/
├── README.md                    # 專案說明
├── SKILL.md                     # 技能入口（AI 讀取用）
├── AI-Editor-新手全能懶人包.md   # 主文件
├── 使用說明.md                  # 本文件
├── .gitignore
└── skills/
    ├── 00-install-all/   一鍵全裝
    ├── 01-environment/   基礎環境（Node.js、Python、PowerShell）
    ├── 02-notebooklm/    NotebookLM 知識庫
    ├── 03-github/        GitHub 版本控制
    ├── 04-netlify/       Netlify 靜態部署
    ├── 05-gas/           Google Apps Script
    ├── 06-workspace/     專案流程自動化（開工/收工）
    ├── 07-obsidian/      Obsidian 第二大腦
    ├── 08-doctor/        系統健康檢查
    ├── 09-upgrade/       一鍵升級
    ├── 10-troubleshoot/  疑難排解
    ├── 11-other-editors/ 其他 AI 編輯器（選用）
    ├── 12-other-deploy/  其他部署平台（選用）
    ├── 13-php-env/       PHP+MySQL 開發必要要求
    └── 14-phpmysql/      PHP+MySQL 專案開發規範（安裝精靈/CRUD/備份/匯入）
```

## 使用方式

### 方式一：直接叫 AI 幫你裝（最簡單）
把這行複製貼給你的 AI 助理：
> 這是我的 AI 編輯器懶人包 https://github.com/tkbooks-2026/ckhotgav-lazy-pack
> 請讀取 repo 內容，列出所有可用的技能，問我要安裝哪些，然後自動幫我安裝。

### 方式二：手動閱讀
開啟 `AI-Editor-新手全能懶人包.md` 或各技能 SKILL.md 參考步驟。

## 本地副本
此資料夾 `Z:\相關參考用資料\ckhotgav-lazy-pack\` 為本地備份，
可隨時從 GitHub 拉取最新版本：
```
git pull https://github.com/tkbooks-2026/ckhotgav-lazy-pack
```

## 安裝 phpmysql 技能到其他 PC

將 `phpmysql` 技能部署到另一台電腦的 OpenCode：

### 方式一：AI 自動安裝（推薦）
在該電腦上對 AI 說：
> 請讀取 https://github.com/tkbooks-2026/ckhotgav-lazy-pack
> 幫我安裝技能 14（phpmysql）到系統技能目錄

### 方式二：手動複製
1. Clone 或下載此 repo
2. 將 `skills/14-phpmysql/` 資料夾複製到 `~\.config\opencode\skills\phpmysql\`
3. 重啟 OpenCode

### 驗證
對 AI 說「phpmysql」，確認有載入 PHP+MySQL 專案開發規範即完成。
