# 一鍵全裝（核心）

依序執行以下核心技能的安裝步驟：

1. 讀取並執行 `skills/01-environment/SKILL.md`
2. 讀取並執行 `skills/02-notebooklm/SKILL.md`
3. 讀取並執行 `skills/03-github/SKILL.md`
4. 讀取並執行 `skills/04-netlify/SKILL.md`
5. 讀取並執行 `skills/05-gas/SKILL.md`
6. 讀取並執行 `skills/06-workspace/SKILL.md`
7. 讀取並執行 `skills/07-obsidian/SKILL.md`
8. 讀取並執行 `skills/08-doctor/SKILL.md`（整體驗證）

每完成一項，請告知使用者目前進度，並等待確認後再繼續。

## 選用項目
以下技能**不包含**在一鍵全裝中，需使用者明確指定才安裝：
- `09-upgrade` 升級既有 CLI
- `10-troubleshoot` 修復問題
- `11-other-editors` Claude Code / Gemini / Codex
- `12-other-deploy` Vercel / Cloudflare / Railway
